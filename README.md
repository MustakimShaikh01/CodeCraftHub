
# CodeCraftHub

CodeCraftHub is a simple learning platform API built with Flask.

It allows developers to track courses they want to learn.

## Features

- Create courses
- View courses
- Update courses
- Delete courses
- JSON file storage
- Simple admin dashboard

## Install

pip install -r requirements.txt

## Run

python app.py

API will run at:

http://localhost:5000

## Dashboard

Open:

dashboard.html

## API Endpoints

GET /api/courses  
POST /api/courses  
GET /api/courses/{id}  
PUT /api/courses/{id}  
DELETE /api/courses/{id}
